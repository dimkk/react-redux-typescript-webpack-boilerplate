# react-redux-typescript-webpack-boilerplate

> A hyper minimalist boilerplate to get started with React/Redux/Typescript2/Babel6/Webpack2

This is just a hyper minimalist boilerplate to get started with:

- React
- React Redux
- React Router
- React Router Redux
- Redux
- Typescript2
- Webpack2

## Installation

```bash
npm install -g yarn
yarn
yarn run build
```
